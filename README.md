# be-a-gambler

웹 브라우저에서 실행할 수 있는 트럼프 카드 점수 게임 웹 앱을 추가했습니다.

## 실행 방법

1. `index.html`을 브라우저에서 열어 플레이하세요.
2. `flipBtn` 버튼을 눌러 카드를 뒤집고 점수를 확인할 수 있습니다.
3. `자동 진행` 버튼으로 자동 게임 진행을 시작할 수 있습니다.
4. `다시 시작` 버튼으로 새 게임을 시작하세요.

## 파일 설명

- `index.html`: 게임 UI
- `styles.css`: 스타일
- `script.js`: 게임 로직
- `main.py`: 기존 터미널용 게임 스크립트
